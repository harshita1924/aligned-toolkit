# Output vs Ability Demonstration

## Scenario
Two students submit equally strong final answers.

### Student A
- Demonstrates reasoning
- Shows draft evolution
- Reflects critically
- Transparently uses AI

### Student B
- Uses AI-generated output directly
- Shows limited understanding
- Provides no reasoning evidence

## Key Insight

Outputs may appear similar while underlying abilities differ significantly.

This demonstrates why output-only assessment is insufficient in AI-supported learning environments.
