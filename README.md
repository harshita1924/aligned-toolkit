# ALIGNED Toolkit

A lightweight open-source toolkit designed to support ability-oriented assessment in AI-supported Learning Management System (LMS) environments.

## Overview

The ALIGNED framework redefines assessment in AI-mediated learning environments by shifting focus from surface-level performance outcomes toward deeper learner abilities.

Traditional LMS assessments primarily evaluate correctness, completion, and final outputs. However, in the age of large language models (LLMs) such as ChatGPT, high-quality outputs no longer reliably indicate authentic learning.

The ALIGNED framework addresses this challenge through a three-layer assessment model:

1. Output Layer
2. Process Layer
3. Ability Layer

An additional Implementation & Governance Layer supports practical adoption.

---

## Framework Layers

### Output Layer
Captures observable performance metrics:
- Correctness
- Completeness
- Writing quality
- Accuracy

### Process Layer
Captures learning behaviors and interaction traces:
- Revision tracking
- Reflection and engagement
- AI usage behavior
- Interaction patterns

### Ability Layer
Captures deeper learner competencies:
- Epistemic Reasoning
- Metacognitive Regulation
- Transfer & Abstraction
- Collaborative-Dialogic Ability
- Ethical AI Judgment

### Implementation & Governance Layer
Supports institutional and technological adoption:
- LMS tools and analytics
- Instructor training
- Institutional policies
- AI governance and transparency
- Resource constraints

## Why This Toolkit?

This toolkit was developed in response to empirical findings showing that:

- Current LMS assessments overemphasize final outputs
- AI-generated content complicates authentic evaluation
- Instructors need process-oriented assessment tools
- Existing LMS environments lack support for reasoning capture and AI transparency
